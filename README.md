# ダウトゲーム with ベッティング - 起動手順

## 📋 必要なもの
- Node.js がインストールされていること

## 🚀 セットアップと起動

### 1. フォルダ構成を確認
```
doubt-betting-game/
├── server.js           # サーバーメイン
├── package.json        # 依存関係
└── public/
    └── index.html      # フロントエンド
```

### 2. 依存パッケージをインストール
ターミナル/コマンドプロンプトでこのディレクトリに移動して実行：
```bash
npm install
```

### 3. サーバーを起動
```bash
npm start
```

コンソールに以下のように表示されます：
```
╔════════════════════════════════════════════════════════════╗
║   🎲 ダウトゲーム with ベッティング                           ║
║   Server running at http://localhost:3000                  ║
║   ブラウザで http://localhost:3000 を開いてください           ║
╚════════════════════════════════════════════════════════════╝
```

### 4. ブラウザで遊ぶ
ブラウザで `http://localhost:3000` にアクセス

## 🎮 ゲームの流れ

1. **プレイヤー数を選択** - 2～6人で遊べます
2. **各プレイヤーがベットを置く** - $1～$100の間で自由に
3. **主張を選ぶ** - 「本当」か「ウソ」かを主張
4. **結果が表示** - オッズに応じて配当が決まります

## 💰 オッズの仕組み（競馬方式）

投票額が少ないほど高いオッズが付きます。
- 総賭け金が大きいプレイヤー → 低オッズ
- 総賭け金が小さいプレイヤー → 高オッズ

## 🛑 サーバーを止めるには
ターミナルで `Ctrl + C` を押す

## 📝 トラブルシューティング

### `npm: command not found` が出る場合
→ Node.js がインストールされていません  
https://nodejs.org/ からインストール

### ポート 3000 が既に使用されている場合
server.js の `const PORT = 3000;` の部分を別の番号に変更してください

### 表示がおかしい場合
→ ブラウザのキャッシュをクリアしてみてください (Ctrl + Shift + Delete)
